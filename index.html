HTML ВЕРСТКА

ЧТО ТАКОЕ И ДЛЯ ЧЕГО ОНА НУЖНА?

Верстка – один из важнейших этапов создания веб-сайта, включает в себя достаточно сложный процесс формирования веб-страниц из изображений, созданных веб-дизайнером.

HTML-верстка сайтов нужна для создания полноценных веб-страниц\сайтов, для последующего размещения оных в сети интернет на всеобщее обозрение. Как правило верстальщик делает свою верстку на основе дизайнерского макета сайта, иногда верстальщик сам создает дизайн сайта по просьбе клиента, исходя из его потребностей.

ФИКСИРОВАННАЯ ВЕРСКА:

Ширина страницы имеет строго фиксированное расположение и не меняется в зависимости от ширины и разрешения экрана. Такая верстка будет хорошо отображаться на компьютере.

АДАПТИВНАЯ ВЕРСТКА:

При адаптивной верстке ваш сайт будет подстраиваться под все разрешения с каких устройств к вам бы не зашел пользователь, и будет удобен для использования не только на компьютере, но и на смартфонах и планшетах.

КРОССБРАУЗЕРНАЯ ВЕРСТКА:

Важнейший тип верстки, который отвечает за то, чтобы ваш сайт одинаково красиво выглядел в разных браузерах, коих немало: Chrome, Firefox, Opere, IE, Safari и другие

FLEX/GRID ВЕРСТКА:

При гибкой верстке с помощью CSS свойств Flex и Grid, стало значительно легче управлять элементами на странице, и быстро создавать удобные и адаптивные страницы.

СЕМАНТИЧЕСКАЯ ВЕРСТКА:

Данный тип верстки берет своё начало от создания стандарта HTML5, теперь можем логически помечать специальные блоки на веб-страницах, помогая поисковикам лучше понимать нашу страницу.

ВАЛИДНАЯ ВЕРСТКА:

Это HTML код, который проходит валидацию (проверку) по стандартам консорциума W3C, который занимается развитием HTML, и создает новые стандарты.
