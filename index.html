HTML ВЕРСТКА

ЧТО ТАКОЕ И ДЛЯ ЧЕГО ОНА НУЖНА?

Верстка – один из важнейших этапов создания веб-сайта, включает в себя достаточно сложный процесс формирования веб-страниц из изображений, созданных веб-дизайнером.

HTML-верстка сайтов нужна для создания полноценных веб-страниц\сайтов, для последующего размещения оных в сети интернет на всеобщее обозрение. Как правило верстальщик делает свою верстку на основе дизайнерского макета сайта, иногда верстальщик сам создает дизайн сайта (как это периодически делаю я) по просьбе клиента, исходя из его потребностей.
